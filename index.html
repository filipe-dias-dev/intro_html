História da Web

Os primeiros computadores surgiram foram Harvard Mark 1 em 1944 criado para gerar cálculos mais extensos, Colossus em 1946 funcionava através de um sistema de criptografia transmitindo através do telégrafo, e Eniac da 1 geração era mais robusto eletrônico digital, e por esquentar muito atraia os mosquitos surgindo então o termo "bug" muito utilizado na programação representando erros. E a internet ela começou com o telégrafo equipamento de comunicação para com outras pessoas, e em 1958 foi passado o primeiro cabo submarino fazendo toda a comunicação da rede.


Grandes Nomes da História

Grace Murray - Uma das primeiras programadoras Mark 1 em 1944.
Joseph Carls Robnett Licklider - Recrutado pela DARPA depois de teorizar sobre uma rede galática de computadores.
Robert E. Kahn – Criador do TCP/IP e apresentou o primeiro e-mail.
Tim Berners Lee - Inventor do WWW, popularizou o HTTP E HTML.
Marc Andreessen – Reinventou o navegador Netscape Navigator feito pelo mosaico.
Bill Gates – Fundador da Microsoft e sua equipe criou a internet explore tornando acessível para todos os computadores.
Steve Jobs – Fundador da Apple concorrente da Microsoft popularizou o smartphone iphones.

O que são Clients?

Clients nada mais é que você consumir uma informação através da internet que fica hospedada no servidor sendo devolvida ao cliente, e nesse processo o navegador gera um cache guardando os dados evitando disparar requisições desnecessárias ao servidor. E os navegadores são programas instalados no computador para interpretar os arquivos que vem do servidor como padrão o w3c. E as aplicações webs são aqueles que não é instalado no seu computador acessados pelo browser.

O que são Server?

Servidores são dispositivos que executa o software, gerenciando a rede de recursos abaixo exemplos de hardwares e softwares em servidores.

Hardwares: ( Apache, Ninginx )

  • Armazenamento (discos rígidos);
  • Memorias;
  • Processadores.

Softwares: ( Windows Server, Oracle, Ubuntu, Fedora, Centos )

  • Arquivos;
  • Segurança (firewall);
  • Streaming;
  • E-mail;
  • WEB.

Estrutura Básica do HTML

De início criamos um arquivo dentro da pasta do seu projeto chamado index.html, sendo a página principal de nossa aplicação web e contem em duas parte citando logo abaixo.

  • Head - cabeça da nossa página são informações pré-carregadas para o usuário como title, links de script.
  • Body - corpo da página que é exibido para o usuário.

Para abrir a nossa página web basta clicar com o botão direito do mouse em index.html e selecionar Open With Live Server abrindo o navegador.

Falando de TAGS podendo ser seus atributos

São informados ao navegador a estrutura do site, abaixo algumas delas:

  • h1 ao h6 : título da página web

  • img : Para incluir uma imagem

  • br : Quebra uma linha do seu texto

  • p : Compõe um parágrafo

  • i : Deixa o texto em itálico

  • strong : deixa o texto em negrito

  • input : define o campo que receberá qualquer caractere

  • mark : marca o texto

  • hr : quebra de linha temática com uma linha na horizontal

  • small : define o tamanho pequeno do texto

  • a : âncora para links

  • blockquote : citação de bloco

  • sub : define o texto com letras menores

Links

Serve para navegar entre uma página e outra com a tag "a" âncora seguido do seu atributo e link do endereço href="www.google.com", podemos também usar outro atributo "target=_self" sendo como padrão abrindo na mesma aba, e "target=_blank" para uma nova aba.

E vamos supor que você quer navegar na sua própria aplicação, criamos um arquivo about.html descrevendo o sobre de sua empresa por exemplo, e na página principal basta definir a tag 'a' com o atributo e o link do seu arquivo about.html ficando da seguinte forma href="about.html"clique aqui para ir até o about sendo direcionado para a página about.
